  <action>
    Server Component ("use client" 없음). props: `{ data: TeamDashboardData; tab: 'daily' | 'weekly' }`.

    **테이블 컬럼 구성:**

    Daily 탭 (DailyRecord):
    - 날짜, 매출, 손익, 이용시간, 이용건수, 가동률 (6컬럼)

    Weekly 탭 (WeeklyRecord):
    - 주차, 매출, 손익, 이용시간, 이용건수, 가동률, 목표 (7컬럼)

    **셀 포맷 (CONTEXT.md 결정):**
    - 매출/손익: `₩${value.toLocaleString()}` — 원 단위 전체 (KPI 카드의 만원 단위와 다름)
    - 손익 음수: `text-red-600 dark:text-red-400` 텍스트 색상 적용
    - 이용시간: 정수 시간과 분으로 분리 → `Xh Ym` 형식
      - 예: 2.5 → `2h 30m`, 3 → `3h 0m`
      - 계산: `Math.floor(hours)` + `Math.round((hours % 1) * 60)`
    - 가동률: `value.toFixed(1) + '%'`
    - 이용건수: `value.toLocaleString() + '건'`
    - 날짜/주차: 원본 문자열 그대로

    **스타일:**
    - shadcn `Table`, `TableHeader`, `TableBody`, `TableRow`, `TableHead`, `TableCell` 사용 (`@/components/ui/table`)
    - 테이블 wrapper div: `overflow-x-auto` (모바일 가로 스크롤)
    - TableHeader: `sticky top-0 bg-background z-10` (헤더 고정)
    - 짝수 행: `even:bg-muted/30` (striped — TableRow에 적용)
    - TableHead: `text-right` (숫자 컬럼), `text-left` (날짜/주차)

    **요약 행 (CONTEXT.md 결정):**
    - 테이블 하단에 합계 행, 평균 행 순서로 2행 추가
    - 합계 계산: revenue, profit, usageHours, usageCount 는 `reduce` 합산
    - 평균 계산: 모든 숫자 컬럼 `sum / records.length` (0으로 나누기 방지: length === 0이면 0)
    - 가동률: 평균만 의미 있음 — 합계 행에는 "-" 표시, 평균 행에는 평균값 표시
    - 요약 행 스타일: `font-bold bg-muted/60` — 일반 행과 명확히 구분
    - 첫 컬럼(날짜/주차) 셀: 합계 행 → "합계", 평균 행 → "평균"
    - Weekly 목표 컬럼: 합계 행 → 합계, 평균 행 → 평균

    **컴포넌트 이름:** `DataTable` (named export)
