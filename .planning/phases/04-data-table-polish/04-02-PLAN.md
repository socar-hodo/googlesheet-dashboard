    **상대 시간 계산 함수 (컴포넌트 내부 로컬 함수):**
    ```
    function getRelativeTime(isoString: string): string
    ```
    - `new Date(isoString)` → Date 객체
    - `Date.now() - date.getTime()` → diffMs
    - 분 단위: diffMs < 60000 → "방금 전"
    - 시간 단위: diffMs < 3600000 → `Math.floor(diffMs / 60000) + '분 전'`
    - diffMs < 86400000 → `Math.floor(diffMs / 3600000) + '시간 전'`
    - 그 이상 → `Math.floor(diffMs / 86400000) + '일 전'`

    **절대 시간 포맷 함수 (컴포넌트 내부 로컬 함수):**
    ```
    function getAbsoluteTime(isoString: string): string
    ```
    - `new Date(isoString)` → Date 객체
    - `toLocaleDateString('ko-KR', { year: 'numeric', month: '2-digit', day: '2-digit' })` — 날짜
    - `toLocaleTimeString('ko-KR', { hour: '2-digit', minute: '2-digit', hour12: false })` — 시각
    - 반환 형식: `"YYYY. MM. DD. HH:mm"` (toLocaleDateString 결과 그대로 + 시각)
    - 예시: "2026. 02. 24. 14:30"

    **렌더링:**
    - 전체 형식: `"{relativeTime} ({absoluteTime})"` — CONTEXT.md 결정
    - 예시: "3시간 전 (2026. 02. 24. 14:30)"

    - 전체: `"마지막 업데이트: 3시간 전 (2026. 02. 24. 14:30)"` 형식
