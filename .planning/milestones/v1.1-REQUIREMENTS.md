- [x] **FILT-02**: 기간 선택 시 KPI 카드, 차트, 테이블이 해당 기간 데이터로 즉시 업데이트된다
  - _Outcome: DashboardContent filteredData useMemo, 전체 컴포넌트에 필터된 데이터 전달. Phase 6-03 완료._

  - _Outcome: URL ?period=last-month 동기화, 새로고침 시 복원. Phase 6-03 완료._

### 내보내기 (Export)

- [x] **EXPO-01**: 사용자가 현재 보이는 데이터를 CSV 파일로 다운로드할 수 있다
  - _Outcome: exportToCsv (UTF-8 BOM, RFC 4180 이스케이프), CSV 버튼 UI. Phase 7-01/02 완료._
- [x] **EXPO-02**: 사용자가 현재 보이는 데이터를 Excel(.xlsx) 파일로 다운로드할 수 있다
  - _Outcome: exportToXlsx (SheetJS 0.20.3 CDN tarball), Excel 버튼 UI. Phase 7-01/02 완료._
- [x] **EXPO-03**: 내보내기 파일명에 현재 탭(Daily/Weekly)과 날짜가 포함된다
  - _Outcome: 파일명 `daily-2026-03-01.csv` / `weekly-2026-03-01.xlsx` 형식. Phase 7-02 브라우저 검증 완료._

### 스파크라인 (Sparkline)

- [x] **SPRK-01**: KPI 카드 각각에 최근 데이터 추이를 보여주는 미니 차트가 표시된다
  - _Outcome: Recharts AreaChart (height=40, 축/툴팁 없음), daily 7일, weekly 8주. Phase 8-01/02 완료._
