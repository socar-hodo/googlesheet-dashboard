# Milestone v1.2: 고객 유형 분析

**Status:** ✅ SHIPPED 2026-03-02
**Phases:** 9-10
**Total Plans:** 4

## Overview

고객 유형(왕복/부름/편도) 이용건수를 도넛 차트와 추이 차트로 시각화하는 분析 기능을 추가했다. v1.1 데이터 레이어 위에 TypeScript 타입·파서·단위 테스트로 데이터 기반을 완성하고, Recharts PieChart + BarChart로 대시보드에 완전히 통합했다.

## Milestone Summary

**Key Decisions:**

- Phase 9 단독 데이터 레이어: CTYPE/REV/COST 세 영역 타입·파서를 한 단계로 묶어 Phase 10/11/12가 병렬 의존 가능하도록 설계
- `parseCustomerTypeFromRows` dateFieldName 파라미터: 일별("일자")/주차별("주차") 구분, 단일 함수로 두 시트 지원
- 고객 유형 파싱은 dailyRows/weeklyRows 재활용: 추가 fetch 없이 기존 일별/주차별 시트에서 처리
- `'${DAILY_RAW_SHEET}'!A1:DZ` 형식: 대괄호 포함 시트명 단일 따옴표 감싸기 필수
- chart3/4/5 oklch 값을 globals.css --chart-3/4/5에서 직접 참조 (CSS 파일 수정 불필요)
- PieChart 중앙 텍스트 x="50%" y="50%": 픽셀 좌표는 ResponsiveContainer 폭 변동 시 깨짐
- stackId 바에서 radius는 최상단 Bar(편도)만 적용: 내부 세그먼트 radius 시 시각적 갭 방지
- customerTypeDaily 날짜 필터는 인라인 date 비교: CustomerTypeRow 타입이 DailyRecord와 달라 filterDailyByPeriod 직접 사용 불가

- `npm run build` 파일 잠금 충돌 (Phase 10-01): `npx tsc --noEmit`으로 타입 검사 대체 — 에러 없음 확인
